# waiting-time-paradox

Python code and LaTeX graphic for my
[post](https://wgunderwood.github.io/2020/07/19/waiting-time-paradox.html)
on the waiting time paradox.

## Usage

* The simulations require Python and can be run with

```
python3 waiting_script.py
```

* The graphic requires LaTeX, TikZ and Imagemagick.
It can be generated with

```
cd time_diagram && bash build.sh
```
